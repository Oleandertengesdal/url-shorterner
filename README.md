# URL Shortener

A full-stack URL shortener application built with Vue 3 frontend and Spring Boot backend.

## Getting Started

### Prerequisites
- Node.js v22.13.0+
- Java 17+
- npm or yarn

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

The frontend will be available at `http://localhost:5173`

### Backend Setup

```bash
cd backend
mvn clean install
mvn spring-boot:run
```

The backend API will be available at `http://localhost:8080`

## Project Structure

- `/frontend` - Vue 3 TypeScript application
- `/backend` - Spring Boot REST API
- `/README.md` - Project documentation
